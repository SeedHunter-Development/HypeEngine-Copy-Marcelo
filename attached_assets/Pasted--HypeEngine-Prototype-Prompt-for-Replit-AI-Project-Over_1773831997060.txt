🚀 HypeEngine Prototype Prompt (for Replit AI) 

🧠 Project Overview 

Build a mobile-first web app prototype called HypeEngine. 

HypeEngine connects: 

Crypto Projects (Clients) → create campaigns using credits 

KOLs (Influencers) → browse campaigns and post on X (Twitter) to earn credits 

The system should feel like a simplified version of Google Ads, but for Twitter/X influencer campaigns. 

Focus on: 

Clean UX 

Fast onboarding 

Minimal friction 

Modern UI (2025 SaaS style) 

 

🎨 Design Requirements 

Mobile-first (primary), responsive for desktop 

Modern SaaS style: 

Soft gradients 

Rounded cards 

Clean typography 

Subtle animations 

Use a component-based layout 

Suggested stack: Next.js  

Light mode default
