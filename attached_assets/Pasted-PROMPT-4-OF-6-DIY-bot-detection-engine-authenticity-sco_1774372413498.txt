TASK: Build the 5-signal bot detection engine.

STEP 1 — BOT DETECTION ENGINE:

Create file: artifacts/hypeengine/lib/matching/authenticity.ts

Function: computeAuthenticityScore(kolProfileId: string): Promise<AuthenticityResult>

5 signals, each normalized to 0-1 (1 = healthy, 0 = likely botted):

SIGNAL 1 — Engagement rate anomaly (weight 0.20):
- Healthy ranges by follower tier:
  <10K: 2-8%, 10K-100K: 1-5%, 100K-1M: 0.5-3%, >1M: 0.3-2%
- Score 1.0 if within range, scale down linearly below minimum
- Penalize if suspiciously HIGH (>15%) — possible engagement pods
- If null, score = 0.5

SIGNAL 2 — Engagement consistency (weight 0.15):
- Healthy CV: 0.4 to 1.5
- Below 0.3 = suspiciously uniform → score drops to 0
- Above 2.0 = one viral hit → score drops to 0.3
- Smooth transition at boundaries
- If null, score = 0.5

SIGNAL 3 — Reply quality (weight 0.30):
- replyQualityScore / 100
- HIGHEST weight — genuine replies nearly impossible to fake
- If null, score = 0.5 and weight becomes 0.15

SIGNAL 4 — Follower sample audit (weight 0.20):
- followerSampleFlag (avg bot score 0-5):
  0-1.0 → authenticity 1.0
  1.0-2.0 → 0.7
  2.0-3.0 → 0.4
  3.0+ → 0.1
- Linear interpolation between breakpoints
- If null, score = 0.5 and weight becomes 0.10

SIGNAL 5 — Growth pattern (weight 0.15):
- Load last 30 days of kol_follower_snapshots
- If fewer than 7 snapshots: score = 0.5, weight = 0.075
- Calculate daily growth, find days where growth > 3x average
- 0 spike days → 1.0, 1 spike → 0.7, 2-3 spikes → 0.4, 4+ → 0.1

COMPOSITE:
authenticityScore = (Σ signal × weight) / (Σ active weights) × 100

Return: { authenticityScore, engRateFlag, consistencyFlag, replyQualityFlag, followerSampleFlag, growthPatternFlag, details: "human-readable summary" }
