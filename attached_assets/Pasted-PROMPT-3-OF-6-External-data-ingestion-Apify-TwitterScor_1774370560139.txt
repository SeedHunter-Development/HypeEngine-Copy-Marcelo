STEP 1 — APIFY INTEGRATION:

Create file: artifacts/hypeengine/lib/data-pipeline/apify.ts

Function: enrichKolFromApify(twitterHandle: string): Promise<ApifyData>

Call Apify Twitter Scraper API to:
a) Scrape last 200 tweets (content, likes, retweets, replies, quotes, date, is_retweet flag)
b) Scrape top 20 replies on last 10 tweets (reply text)
c) Scrape 300 follower profiles (bios, locations, follower counts, tweet counts, usernames)

Use Apify API: base URL https://api.apify.com/v2, use "apidojo/tweet-scraper" actor or equivalent, Bearer token from APIFY_API_TOKEN.

From scraped data, COMPUTE locally:
- twitterFollowers, twitterFollowing: from profile
- avgPostsPerDay: tweets in date range / days
- avgLikes: mean likes on non-RT tweets
- avgRetweets: mean retweets
- avgReplies: mean replies
- engagementRate: (avgLikes + avgRetweets + avgReplies) / twitterFollowers
- rtToLikeRatio: avgRetweets / avgLikes
- quoteTweetRatio: quote tweets / total
- originalVsRtRatio: original tweets / total
- threadFrequency: self-reply tweets / total
- engagementConsistency: stddev(likes) / mean(likes)
- followerSampleGeo: parse locations from 300 bios → country percentages
- followerSampleFlag: for each follower, score bot signals (no avatar +1, no bio +1, random username +1, <5 tweets +1, following/follower >10 +1), average across all

IMPORTANT: If APIFY_API_TOKEN not set, return null and log warning. Never crash.

STEP 2 — TWITTERSCORE INTEGRATION:

Create file: artifacts/hypeengine/lib/data-pipeline/twitterscore.ts

Function: enrichKolFromTwitterScore(twitterHandle: string): Promise<TwitterScoreData>

Call TwitterScore API (base URL https://api.twitterscore.io, key from TWITTERSCORE_API_KEY).

Extract:
- twitterScoreValue: main score
- notableFollowers: array with type (VC/exchange/project/influencer)
- vcFollowerCount: count of VC followers

Same graceful fallback if API key missing.
